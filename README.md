# Software AI Bot

一个基于 Electron 和 React 的智能聊天应用，具有以下特性：

- 支持多种 AI 模型选择（GPT-3.5 Turbo, GPT-4, Claude 2）
- 对话历史管理功能
- 对话标题编辑
- 历史记录搜索
- 新建对话功能

## 技术栈

- Electron
- React
- Material-UI
- Webpack

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建应用
npm run build
```
